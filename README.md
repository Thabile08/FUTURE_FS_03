# 🌿 Heavenly Relaxation Spa Website

A premium, mobile-friendly business website for **Heavenly Relaxation Spa**, designed to showcase services, pricing, and brand identity while providing an easy way for clients to book appointments.

---

## ✨ Overview

This website was built using **HTML, CSS, and JavaScript** with a focus on:

* Luxury aesthetic (black, gold and purple minimal design)
* Mobile-first responsiveness
* Clean user experience
* Easy booking via WhatsApp
* Strong brand presentation (CEO section, interviews, services)

---

## 🚀 Features

### 🧭 Navigation

* Fixed responsive navigation bar
* Smooth scrolling between sections

### 🖼️ Hero Section

* Full-screen background
* Clear branding message
* Call-to-action button

### 💅 Services Section

* Clickable service cards
* Dynamic image preview when a service is selected

### 💰 Pricing Section

* Organized column layout
* Responsive (stacks on mobile, columns on desktop)
* Highlighted pricing for readability

### 👩‍💼 CEO Section

* Personal branding for trust and credibility
* Responsive layout with image + text

### 🎤 Interviews & Features

* Showcases media appearances and business credibility
* Optional links to videos or articles

### 🖼️ Gallery

* Horizontal scroll gallery of work

### 📱 WhatsApp Booking

* Direct booking link
* Floating WhatsApp button for quick access

### 🌐 Social Media Integration

* Instagram, Facebook, WhatsApp icons
* Brand-colored hover effects with premium styling

---

## 🎨 Design Philosophy

This website follows a **luxury minimal design approach**:

* Limited color palette (black, gold, white)
* Clean spacing and layout
* Subtle animations instead of flashy effects
* Focus on readability and user experience

---

## 🛠️ Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling and responsiveness
* **JavaScript (Vanilla)** – Interactivity
* **Font Awesome** – Social media icons
* **Google Fonts** – Typography

---

## 📱 Responsiveness

The website is fully responsive:

* 📱 Mobile (primary focus)
* 📲 Tablet
* 💻 Desktop

Layouts automatically adjust using CSS Grid and media queries.

You can easily customize:

* Images → replace inside `/images`
* Prices → edit in pricing section
* Contact details → update phone/email
* Social links → replace URLs in footer
* CEO info → update image and text
